## 🚀 How to Publish on GitHub Pages

### Step 1 — Create a GitHub Repository
1. Go to [github.com](https://github.com) and log in
2. Click **+** → **New repository**
3. Name it: `itsuniqueplayz-portfolio` *(or any name you like)*
4. Set visibility to **Public**
5. Click **Create repository**

### Step 2 — Upload the Files
**Option A — Via GitHub Web UI (easiest):**
1. Open your new repository
2. Click **Add file** → **Upload files**
3. Drag & drop **all files** from this folder
4. Click **Commit changes**

**Option B — Via Git (command line):**
```bash
git init
git add .
git commit -m "Initial portfolio deploy"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/itsuniqueplayz-portfolio.git
git push -u origin main
```

### Step 3 — Enable GitHub Pages
1. Go to your repository → **Settings**
2. Scroll to **Pages** (left sidebar)
3. Under **Source**, select **Deploy from a branch**
4. Choose branch: **main** · folder: **/ (root)**
5. Click **Save**
6. Wait ~1-2 minutes, then visit your live URL!
